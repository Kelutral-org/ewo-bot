```
Spiel
```
Das Wortspiel ist ein auf Wörtern basiertes Spiel, das entwickelt wurde, um das Vokabular, die Geschwindigkeit und Verständnis des Lautsystems der Sprache zu verbessern.
Das genaue Ziel des Spiels basiert auf dem Spielmodus, aber das Grundprinzip ist universell: Nenne ein Wort, das mit demselben Buchstaben beginnt, auf den das vorangegangene Wort geendet ist.
Zum Beispiel:
``plltxe``
``eltu``
``utral``
usw.

```
Spielmodi
```
Das Wortspiel hat zwei Modi: Einzel- und Mehrspieler:
**Einzelspieler:**
    ・Der Bot wird automatisch antworten bzw. mitspielen.
    ・Nur der Spieler, der das Spiel gestartet hat, kann spielen.
    ・Punkte können nicht gesammelt werden.

**Mehrspieler:**
    ・Der Bot wird nicht antworten bzw. mitspielen.
    ・Mehr als ein Spieler wird benötigt.
    ・Punkte können gesammelt werden.

```
Schwierigkeitsstufen
```
Das Wortspiel hat drei Schwierigkeitsstufen: Casual, Wettkampf oder Deathmatch.
**Casual:**
    ・Wörter können wiederverwendet werden, jedoch nicht zu häufig (fünf andere Wörter müssen zwischen der Nennung desselben Wortes genannt werden, um es erneut nennen zu können).
    ・Das Spiel wird nicht enden, solange man es nicht manuell beendet.
    ・Punkte werden an jeden Mitspieler verteilt, und Boni werden nicht vergeben.

**Wettkampf**:
    ・Wörter können nicht wiederverwendet werden.
    ・Der Spieler, der zuerst ein Wort nennt, das keine Optionen zum Weiterspielen offen lässt, gewinnt.
    ・Punkte werden an jeden Mitspieler verteilt, der Gewinner erhält jedoch einen Bonus.

**Deathmatch**:
    ・Du musst dem Spiel beitreten, wenn es beginnt.
    ・Wenn ein Spieler ein Wort nennt, das bereits verwendet wurde, wird er aus dem Spiel ausgeschlossen.
    ・Der letzte Spieler, der übrig bleibt, gewinnt.
    ・Punkte werden an jeden Mitspieler verteilt, der Gewinner erhält jedoch einen Bonus.
    ・Erzwingt den Mehrspieler-Modus.
